# Concertly Mobile

A mobile application that helps users discover concerts, follow artists, share concert experiences, and connect with other music fans.

## Features

- User authentication
- Concert discovery
- Artist profiles
- Venue information
- Social feed
- Concert sharing
- Personalized recommendations

## Tech Stack

- React Native / Expo
- TypeScript
- PostgreSQL

## Purpose

Concertly aims to bring concert-goers, artists, and live events together in a single social platform.
